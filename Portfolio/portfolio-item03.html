Coded by: James Hicks

VideoActive Games was a PHP storm project aimed at achieving CRUD methods to enable functional usability. The site replicates a video game shopping website. My responsibilities included building a search feature, debugging code, and co-leading the project. The main focus of the project was functionality.

VideoActive Games connects user entered data to a MySQL Database and updates data in real time. Users can sign in, add/remove items from a shopping cart and when logged in as an admin the admin can edit, update, delete, or add new games.
