Travelfly

Coded by: James Hicks

Travelfly was my first attempt at a responsive website. To achieve responsiveness I had to use media queries with a max-width value in order to change pixel size and change the styling of divs in side by side.

Travelfly was a single page website that I worked on over the course of 6 weeks. The goal of the project was to make a single paged responsive website that was user friendly in design and navigation.

Some of the issues I encountered in this project were alignment and styling issues, figuring out how to style the right way and functionality with JavaScript. Using JavaScript to make navigating the site possible was a challenge that I overcomed.

Here is the link to view the site on github: https://hicksjc.github.io/Travelfly/Travelfly-main/Travelfly/
