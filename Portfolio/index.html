Responsiveness

Throughout these few development projects responsiveness and design has been a priority for some. The ability to make a website responsive to the changing of a viewport size isn't easy but when it is done it can be not only refreshing for the programmer but also easier to understand for the viewer. It allows the viewer to be able to increase the size of text to see better, and changes several aspects of the site itself to compensate for a smaller or larger viewport size.
